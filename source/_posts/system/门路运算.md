# 1. 说明

```
与   同真为真  一假为假     &=
或   同假为假  一真为真     |=
                         ^ (取反?)
异或 值相同为真 值不相同为假 
```


```go
// Sets the bit at pos in the integer n.
func setBit(n int, pos uint) int {
    n |= (1 << pos)
    return n
}

// Clears the bit at pos in n.
func clearBit(n int, pos uint) int {
    mask := ^(1 << pos)
    n &= mask
    return n
}

反转
func reserve1Bit(n byte, pos uint32) byte{
	if pos > 7 {
		return 0
	}
	m := byte(1 << pos)
	m &= n
	if m == 0 {
		n |= byte(1 << pos)
		return n
	} else {
		n &= ^(1 << pos)
		return n
	}
}
```
